# Contributing to Novus

Thanks for contributing.

## Before You Start

1. Read the README and open issues.
2. Comment on an issue before starting significant work.
3. Ask if the issue is unclear.
4. Do not begin large architecture changes without discussion.

## Workflow

```text
Issue
→ assign/comment
→ create branch
→ implement
→ test
→ open pull request
→ review
→ merge

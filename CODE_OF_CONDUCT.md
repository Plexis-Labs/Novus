# Code of Conduct

Novus is a collaborative student open-source organization.

We expect contributors to be respectful, constructive, and honest about their work.

## Expected Behavior

- communicate clearly
- give useful code review
- respect different skill levels
- credit others properly
- raise concerns early
- focus criticism on ideas and code, not people

## Unacceptable Behavior

- harassment or personal attacks
- claiming other people’s work
- sharing private credentials or personal information
- deliberately breaking project infrastructure
- repeated disrespectful behavior

For concerns, contact a founding owner privately.
